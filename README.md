# 🛒 Cart-Flow (Frontend)

Cart-Flow is a modern full-stack marketplace application where users can discover products through search and category filtering, while sellers can create and manage their own product listings. This project is built with **Next.js** and **TypeScript**, focusing on a clean UI, responsive design, and smooth user experience.

> 🚀 **My first project built with TypeScript.**

## 🌐 Live Demo

**Live Website:** https://cart-flow-henna.vercel.app

---

## ✨ Features

- 🔍 Search products by name
- 📂 Browse products by category
- 📱 Fully responsive design
- 🛍️ View detailed product information
- ➕ Sell new products
- ✏️ Update your own products
- 🗑️ Delete your own products
- 👤 Seller dashboard for managing listings
- 🎨 Modern and clean UI
- ⚡ Fast performance with Next.js

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js (v18 or later)
- npm, yarn, pnpm, or bun

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/cart-flow-frontend.git
```

Go to the project directory:

```bash
cd cart-flow-frontend
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file in the root directory and add your environment variables:

```env
NEXT_PUBLIC_API_URL=YOUR_BACKEND_URL
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
